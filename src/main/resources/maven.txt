clean - очистка от target
validate - проверяет корректность метаинформации о проекте
compile - компилирует исходники проекта
test - запускает все тесты из папки test
package - упаковывает скомпилированные классы в новый артефакт
verify - проверяет корректность артефакта + проверка качества
install - добавляет артефакт в локальный репозиторий
site - перезакачивает библиотеки
deploy - заливает артефакт на production-server или удалённый репозиторий


mvn versions:display-dependency-updates    команда обновляет версии
mvn clean install    можно сразу задавать несколько команд
